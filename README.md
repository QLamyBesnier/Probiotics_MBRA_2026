# Data and code corresponding to the analysis of:

_Lamy-Besnier et al. (2026) in vitro modeling of the intestinal microbiota predicts individualized probiotic response in metabolic syndrome_

### Data

The data necessary to reproduce the analysis can be found in the "Data" folder.

It is composed of 17 files:

- *Donors_MBRA_Probiotic.csv*: a csv table containting metabolic data from the donors used in the MBRA experiment.
- *B_animalis_MBRA_Probiotic.csv*: a csv table containting B. animalis qPCR data.
- *L_plantarum_MBRA_Probiotic.csv*: a csv table containting L. plantarum qPCR data.
- *L_reuteri_MBRA_Probiotic.csv*: a csv table containting L. reuteri qPCR data.
- *16S_qPCR_MBRA_Probiotics.csv*: a csv table containing the 16S qPCR data.
- *Shannon_norm3_MBRA_Probiotic.csv*: a csv table containting .
- *Shannon_raw_MBRA_Probiotic.tsv*: a tsv table containting .
- *BC_norm3_MBRA_Probiotic.csv*: a csv table containting normalized Bray-Curtis data.
- *Distance_matrix_MBRA_Probiotic.tsv*: a tsv table containting the distance matrix (Bray-Curtis distances).
- *TLR4_MBRA_Probiotic.csv*: a csv table containting LPS data.
- *TLR5_MBRA_Probiotic.csv*: a csv table containting flagellin data.
- *Genus_relab_MBRA_Probiotic.csv*: a csv table containting genus level relative abundances per sample.
- *Metadata_Maaslin_MBRA_Probiotic.csv*: a csv table containting metadata for all the sequenced samples, adapter for Maaslin3.
- *Metadata_MBRA_Probiotic.txt*: a tsv table containting metadata for all the sequenced samples.
- *Multivariate_T10_MBRA_Probiotic.csv*: a csv table containting data for the multivariate analysis.
- *ROC_T10_Control_MBRA_Probiotic.csv*: a csv table containting data for the ROC analysis on the control chambers.
- *ROC_T10_Probiotics_MBRA_Probiotic.csv*: a csv table containting data for the ROC analysis on the probiotic-treated chambers.

### Code

The code corresponding to the data analysis can be found in the "Code" folder.

It is composed of a single RMarkdown file, which generates all the corresponding figures found in the manuscript.
